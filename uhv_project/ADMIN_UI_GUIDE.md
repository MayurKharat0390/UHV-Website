## 📊 **Common Admin Tasks**

### **Daily Tasks:**
1. **Approve Impact Stories**
   - Go to: Voices → Impact Stories
   - Select pending stories
   - Actions → "Approve selected stories"

2. **Monitor Activity**
   - Go to: Progress → User Progress
   - View streak charts
   - Check engagement stats

### **Weekly Tasks:**
1. **Add Reflection Scenarios**
   - Go to: Reflections → Reflection Scenarios
   - Click "Add Reflection Scenario"
   - Fill in details + 4 options
   - Set active date

2. **Create Activities**
   - Go to: Activities → Activities
   - Click "Add Activity"
   - Upload image, set date

3. **Add Resources**
   - Go to: Resources → Resources
   - Choose type (Article/Video/PDF/Link)
   - Upload files or add URLs

### **Monthly Tasks:**
1. **Add New Quotes**
   - Go to: Quotes → Quotes
   - Add inspirational quotes
   - Categorize by value

2. **Update Faculty**
   - Go to: Faculty → Faculty Profiles
   - Update bios and photos

## 💡 **Pro Tips**

### **Quick Search**
- Use the search bar (top right) to find anything quickly
- Search by username, email, title, etc.

### **Filters**
- Use right sidebar filters to narrow down results
- Combine multiple filters
- Date range filtering available

### **Bulk Actions**
- Select multiple items (checkboxes)
- Choose action from dropdown
- Click "Go"

### **Inline Editing**
- When adding Reflection Scenarios, add all 4 options inline
- No need to create options separately

### **View Site Link**
- Click "View Site" in top menu to see public site
- Opens in new tab
